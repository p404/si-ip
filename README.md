# SI-IP

SI-IP Is a A Dynamic DNS updater built on top of multiple DNS Providers e.g. AWS Route 53.

## Installing
```bash
1. git clone https://github.com/p404/si-ip.git
2. cd si-ip
3. pip install -r requirements.txt
```
